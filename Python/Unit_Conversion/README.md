# Meter to Centimeter Converter

A simple command-line utility that converts distance in meters to centimeters.

## Features
- **Accurate Conversion:** Multiplies the input meter value by 100 to provide the result in centimeters.
- **Input Flexibility:** Accepts both whole numbers and decimal values.

## How to Run

1. Ensure you have Python installed.
2. Open your terminal in the directory where `converter.py` is saved.
3. Run the following command:
   ```bash
   python converter.py
