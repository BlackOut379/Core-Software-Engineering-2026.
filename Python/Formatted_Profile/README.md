# User Profile Generator

A simple Python script that collects user information (first name, last name, and location) and formats the output cleanly.

## Features
- **Data Cleaning:** Automatically handles extra spaces using `.strip()`.
- **Formatting:** Capitalizes names and locations using `.title()` (e.g., "mumbai" becomes "Mumbai").

## How to Run

1. Ensure you have Python installed.
2. Clone this repository or download the `profile.py` file.
3. Open your terminal and run:
   ```bash
   python profile.py
